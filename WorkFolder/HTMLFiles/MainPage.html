Welcome to Animal Database!

This project is an animal database that a user can either make an account to log in so that they can create and interact with the animal or browse as a guest to view the animals. The logged in user will also be a ble to play with and feed the animals. When looking at a list of animals you should be able to sort the listed animals by one of the fields used to make the animal.
